How it works

A complete prediction pipeline accessible through a single conversation interface.

Text-first

Start with a plain question. Attach PDFs, Markdown, or text files only when you have supporting data.

Multi-agent orchestration

Graph building, simulation, and report generation run as separate agents behind a single interface.

Structured result cards

Every simulation produces a structured result card with summary, full report, and a follow-up chat thread.
